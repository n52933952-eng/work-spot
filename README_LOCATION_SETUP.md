# Company Location Setup

## الموقع الرئيسي للشركة

تم تعيين موقع الشركة الرئيسي إلى:
- **الموقع**: الجامعة الأردنية، عمان، الأردن
- **الإحداثيات**: 32.014206, 35.873015
- **الرابط**: https://maps.app.goo.gl/1yscqMESQ9JPEMTb8?g_st=iw
- **نصف القطر**: 50 متر

## إعداد الموقع الافتراضي

### الطريقة 1: عبر API (للمطورين)

بعد تسجيل الدخول كـ Admin، يمكن إنشاء الموقع الافتراضي:

```bash
POST /api/locations/default
Authorization: Bearer YOUR_TOKEN
```

### الطريقة 2: عبر Script (للإعداد الأولي)

```bash
npm run setup:location
```

هذا سيقوم بـ:
1. إنشاء الموقع الرئيسي في قاعدة البيانات
2. تعيين الموقع للموظفين الذين ليس لديهم موقع محدد
3. تفعيل الموقع تلقائياً

## تحديث الموقع

إذا أردت تغيير موقع الشركة، يمكنك:

1. **عبر Admin Panel** (لاحقاً):
   - الذهاب إلى إدارة المواقع
   - تعديل الموقع الرئيسي

2. **عبر API**:
```bash
PUT /api/locations/:locationId
{
  "latitude": 32.014206,
  "longitude": 35.873015,
  "radius": 50
}
```
